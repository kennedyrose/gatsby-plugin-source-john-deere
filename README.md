# gatsby-plugin-source-john-deere

WIP

Use the John Deere API as a Gatsby data source

## Installation

With npm:

```bash
npm install --save gatsby-plugin-source-john-deere
```

Or Yarn:

```bash
yarn add gatsby-plugin-source-john-deere
```

## Usage

```javascript
// In your gatsby-config.js
plugins: [
	'gatsby-plugin-source-john-deere',
]
```
